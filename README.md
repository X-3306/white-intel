# white-intel
Prosty skrypt pomocny do bug bounty i innych czynności OSINT

# Jak uruchomić?

- chmod x+ white_intel.sh

 - ./white_intel.sh -h

# przykład użycia: 
./white_intel.sh  -s Google.com 


# Obecne funkcje:

 - Sprawdź rekordy DNS
 - Pobierz informacje Whois
 - Sprawdź adres IP
 - Szukaj subdomen
